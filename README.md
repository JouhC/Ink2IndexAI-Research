# Ink2IndexAI Research

Ink2IndexAI Research contains research-oriented work for the Ink2IndexAI document-intelligence system. The repository is used to explore model behavior, validation workflows, and experiments that support the production-facing application.

## Focus Areas

- Document AI experimentation.
- Model validation and analysis workflows.
- Research notebooks and supporting assets.
- Iteration around scanned document understanding.

## Related Project

The application repository is [`JouhC/Ink2IndexAI`](https://github.com/JouhC/Ink2IndexAI).
